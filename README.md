# Welcome to the Jacko Moto Mobile Application

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

⚠️ The build of this project has not yet been generated

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Open the app

   - Download the application named expo go on the store
   - Scan the QR Code
